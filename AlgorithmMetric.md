# Algorithm Metrics #

## Data Types ##

### MetricsCommandMetaData ###
Stores metadata information that might be generally useful

- userName
- hostName
- directory
- startedDate
- started
- finishedDate
- elpasedMilliseconds

### MetricManager ###

- statisticsType: Mean, Median, etc
- percentiles: Array
- recordingRanks: position@1,10,100
